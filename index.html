How to Run the Programs

Local terminal · Python 3
  • 1. Clone the repo
    Open a terminal and run:
    git clone https://github.com/selvarajane/4.2.26task.git
  • 2. Run the menu program
    Inside the project folder:
    # Windows (PowerShell / CMD)
    python python_exercises.py
    This opens a menu where you can choose any task number (1–50) and see its output directly in the terminal.
  • 3. Why you don't see Python I/O here
    Vercel hosts this static HTML page. The interactive parts of your project (inputs, loops, ATM, calculator, etc.) are written in Python and therefore run on your machine, not in the browser.
